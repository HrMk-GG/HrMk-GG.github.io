# Voxel Craft

Three.js で作った、ブラウザで動くマインクラフト風のボクセルゲームです。
ビルド不要・依存パッケージのインストール不要（Three.js は CDN から読み込みます）。

## 遊び方

ES Modules を使っているため `index.html` を直接ダブルクリックしても動きません。
簡易HTTPサーバーで配信してください。

```bash
# Python がある場合
python3 -m http.server 8000

# Node.js がある場合
npx serve .
```

ブラウザで `http://localhost:8000` を開き、画面をクリックしてスタートします。

### 操作方法

| キー | 動作 |
|---|---|
| W / A / S / D | 移動 |
| Space | ジャンプ |
| Shift | ダッシュ |
| マウス移動 | 視点操作 |
| 左クリック | ブロックを壊す |
| 右クリック | ブロックを設置 |
| 1〜7 / マウスホイール | 設置するブロックを選択 |
| Esc | ポインターロック解除（メニューに戻る） |

## 機能

- Perlin風ノイズによる自然な地形生成（草原・砂浜・水面）
- ランダムに生成される木
- ブロックの破壊・設置（草・土・石・砂・丸太・葉・板材）
- ボクセル単位のDDAレイキャストによる正確なブロック選択
- 簡易物理演算（重力・ジャンプ・当たり判定）
- チャンク単位でのメッシュ最適化（見えない面は描画しない）

## ファイル構成

```
index.html          エントリーポイント
css/style.css        UIスタイル
js/main.js            シーン構築・ゲームループ
js/world.js           ワールド生成・チャンクメッシュ構築
js/player.js          プレイヤー操作・物理・レイキャスト
js/blocks.js          ブロック定義・テクスチャ生成
js/noise.js           地形生成用ノイズ関数
```

## カスタマイズ

- `js/world.js` の `CHUNKS_X` / `CHUNKS_Z` でワールドサイズを変更できます。
- `js/world.js` の `heightAt()` でノイズのスケールを変えると地形の起伏を調整できます。
- `js/blocks.js` にブロックを追加すれば新しいブロック種類を増やせます。

## 既知の制限

- ワールドは有限サイズ（デフォルト 96×96）で、無限に生成される訳ではありません。
- セーブ/ロード機能はありません（リロードすると地形は再生成されます）。
- モバイル（タッチ操作）には対応していません。
